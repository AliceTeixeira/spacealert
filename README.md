# 🚀 SpaceAlert - Monitoramento de Desastres Naturais

## Descrição do Projeto
O SpaceAlert é uma plataforma web que utiliza dados de satélite da NASA e ESA para monitorar riscos de enchentes, deslizamentos e queimadas no Brasil. O objetivo é fornecer alertas precoces para comunidades em áreas vulneráveis, ajudando a salvar vidas e reduzir danos materiais.

O sistema apresenta um mapa de risco atualizado, lista de alertas ativos e um formulário para cadastro de interessados. O projeto foi desenvolvido para a Global Solution 2026/1 da FIAP, conectando exploração espacial com problemas reais da sociedade.
